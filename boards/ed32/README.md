# ED32 板级支持

本目录按国产 MCU / 替代 MCU 学习对象规划，用于记录兼容性验证、外设差异和迁移过程。

> 如果实际使用的是 GD32，可以将该目录重命名为 `gd32/`。当前型号参数必须以实际芯片丝印、数据手册和开发板原理图为准。

## 芯片索引

| 目录 | 定位 | 重点实验 |
|---|---|---|
| [`ed32f103/`](./ed32f103/) | 国产替代 MCU 学习对象，用于研究 F103 级平台的兼容性和迁移方法。 | GPIO、UART、Timer、ADC、PWM、I2C、SPI 和库迁移验证。 |
| [`ed32f407/`](./ed32f407/) | 国产高性能替代 MCU 学习对象，关注实时控制和外设兼容性。 | FreeRTOS、CAN、DMA、编码器、电机控制和性能对比。 |
| [`ed32vf103/`](./ed32vf103/) | RISC-V MCU 学习对象，重点记录工具链、中断和外设生态差异。 | RISC-V 启动、GPIO、UART、Timer、中断、RTOS 移植和兼容性测试。 |

## 维护要求

- 芯片参数和外设能力必须以对应料号数据手册为准。
- 具体开发板晶振、LED、按键和接口引脚记录在芯片目录文档中。
- 可运行工程放入 `project_template/` 或后续独立项目目录。
- 公共接口优先遵循 [通用 BSP 接口](../common/bsp_interface.md)。
