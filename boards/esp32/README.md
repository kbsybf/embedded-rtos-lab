# ESP32 板级支持

本目录维护 ESP32 系列模组和开发板资料，重点覆盖无线通信、FreeRTOS、边缘感知与机器人交互。


## 芯片索引

| 目录 | 定位 | 重点实验 |
|---|---|---|
| [`esp32-wroom-32/`](./esp32-wroom-32/) | 经典无线 MCU 模组，适合网络连接、实时任务和机器人遥控。 | Wi-Fi、蓝牙、FreeRTOS、MQTT、HTTP、WebSocket 和机器人遥控。 |
| [`esp32-s3/`](./esp32-s3/) | 面向 AIoT 与人机交互，具备更强的向量指令和 USB 能力。 | 摄像头、语音、USB、边缘推理、Wi-Fi 和人机交互。 |
| [`esp32-c3/`](./esp32-c3/) | 低成本 RISC-V 无线 MCU，适合轻量 IoT 与分布式传感节点。 | Wi-Fi、BLE、低功耗、MQTT、传感器采集和 OTA。 |
| [`esp32-cam/`](./esp32-cam/) | 低成本摄像头开发板，适合轻量视觉采集与无线图像传输。 | 摄像头驱动、JPEG 采集、HTTP 推流、Wi-Fi 图像传输和远程监控。 |

## 维护要求

- 芯片参数和外设能力必须以对应料号数据手册为准。
- 具体开发板晶振、LED、按键和接口引脚记录在芯片目录文档中。
- 可运行工程放入 `project_template/` 或后续独立项目目录。
- 公共接口优先遵循 [通用 BSP 接口](../common/bsp_interface.md)。
