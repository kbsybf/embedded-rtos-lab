# ESP32-C3

## 芯片定位

低成本 RISC-V 无线 MCU，适合轻量 IoT 与分布式传感节点。

## 适合实验

Wi-Fi、BLE、低功耗、MQTT、传感器采集和 OTA。

## 推荐用途

低成本 IoT 节点、机器人辅助传感器和无线执行端。
