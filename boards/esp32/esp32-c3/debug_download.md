## 验证步骤

1. 读取芯片 ID 或连接目标。
2. 全片擦除后下载最小点灯工程。
3. 验证断点、单步、变量观察和复位。
4. 记录 Release / Debug 配置差异。

## 常见问题

- 无法连接：检查供电、地线、启动模式、复位和调试接口复用。
- 下载后不运行：检查时钟、链接地址、看门狗和启动文件。
- 芯片锁死：降低调试速度，尝试复位下连接或全片擦除。
- 串口无输出：核对时钟、引脚复用、电平和波特率。
