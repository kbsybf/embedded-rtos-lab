# 启动说明

## 启动流程

1. 上电或复位，处理启动模式与向量表入口。
2. 设置初始栈指针并进入复位处理函数。
3. 完成数据段复制、BSS 清零和系统时钟基础初始化。
4. 初始化 C / C++ 运行时后进入 `main` 或 RTOS 启动入口。

## 启动文件

- 启动文件名称与来源：待填写。
- System 初始化文件：待填写。
- 中断向量表位置：待填写。

## 链接脚本

- GCC linker script / scatter file：待填写。
- Flash、SRAM 和特殊 RAM 分区：待填写。
- Bootloader 存在时的应用偏移：待填写。

## 注意事项

- 中断向量表重定位后同步检查 VTOR 或平台等效配置。
- C++ 工程需确认静态对象初始化和异常 / RTTI 策略。
- FreeRTOS 工程需核对 PendSV、SysTick 和 SVC 优先级。
