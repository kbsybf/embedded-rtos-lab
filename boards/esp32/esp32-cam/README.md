# ESP32-CAM

## 芯片定位

低成本摄像头开发板，适合轻量视觉采集与无线图像传输。

## 适合实验

摄像头驱动、JPEG 采集、HTTP 推流、Wi-Fi 图像传输和远程监控。

## 推荐用途

机器人视觉辅助节点、低成本图传和环境监控。
