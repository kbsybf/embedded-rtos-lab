# ESP32-S3

## 芯片定位

面向 AIoT 与人机交互，具备更强的向量指令和 USB 能力。

## 适合实验

摄像头、语音、USB、边缘推理、Wi-Fi 和人机交互。

## 推荐用途

视觉交互节点、语音控制器、智能终端和机器人交互模块。
