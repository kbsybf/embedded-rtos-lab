# 存储布局

| 区域 | 地址 / 大小 | 用途 |
|---|---|---|
| Flash | 取决于具体模组；起始地址待确认 | 程序代码与只读数据 |
| SRAM | 512 KB 片上 SRAM，部分模组带 PSRAM；分区待确认 | 栈、堆、全局变量与 DMA 缓冲 |
| 系统存储区 | 待查数据手册 | ROM Bootloader 或系统服务 |
| Bootloader | 默认未分配 | 后续按升级方案规划 |
| 应用程序区 | 待填写 | 主固件镜像 |
| 参数区 | 待填写 | 校准参数、设备配置和版本信息 |
| 日志 / 备份区 | 可选 | 故障记录或升级回滚 |

## 维护说明

- 分区变化必须同步链接脚本、下载配置和升级工具。
- 擦写参数区时考虑寿命、掉电保护和完整性校验。
- 带 Cache、MPU、外部 SDRAM 或 PSRAM 的平台需补充一致性约束。
