# STM32F407VET6

## 芯片定位

进阶经典 MCU，性能和外设资源均衡，适合机器人底层控制。

## 适合实验

FreeRTOS、CAN、电机控制、编码器、DMA、ADC、PWM 和多任务通信。

## 推荐用途

机器人主控、移动底盘控制器、电机驱动控制和实时数据采集。
