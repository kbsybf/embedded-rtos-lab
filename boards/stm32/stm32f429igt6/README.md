# STM32F429IGT6

## 芯片定位

外设资源丰富，适合图形界面、外部存储和复杂工业控制实验。

## 适合实验

LCD、LTDC、SDRAM、FMC、DMA、Ethernet 和多外设协同。

## 推荐用途

带显示的人机界面、复杂数据采集终端和工业控制器。
