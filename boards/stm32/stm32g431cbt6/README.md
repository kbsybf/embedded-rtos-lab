# STM32G431CBT6

## 芯片定位

面向混合信号与电机控制的 MCU，适合高精度采样和控制。

## 适合实验

FOC、互补 PWM、ADC 同步采样、运放、比较器和电流环控制。

## 推荐用途

无刷电机控制器、机器人关节驱动和高响应执行器控制。
