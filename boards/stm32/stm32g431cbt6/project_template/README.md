# STM32G431CBT6 最小工程模板

本目录用于保存可编译、可下载的板级最小工程骨架。

| 目录 | 用途 |
|---|---|
| `Core/` | 应用入口、板级初始化和中断处理 |
| `Drivers/` | 厂商驱动、设备驱动和 BSP 实现 |
| `Middlewares/` | FreeRTOS、协议栈和第三方中间件 |

- [ ] 串口日志工程
