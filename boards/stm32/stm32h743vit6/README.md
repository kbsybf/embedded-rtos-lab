# STM32H743VIT6

## 芯片定位

高性能 MCU，适合高实时性、多传感器和复杂机器人控制任务。

## 适合实验

高速 ADC、DMA、Cache、Ethernet、FDCAN、多传感器融合和高频控制环。

## 推荐用途

高性能机器人控制器、实时融合节点和多轴运动控制。
